# Evaluation Rubric

Rate each dimension 1-5:

## Factual Accuracy (1-5)
- 5: Completely accurate, verifiable
- 3: Mostly accurate, minor errors
- 1: Contains significant factual errors or hallucinations

## Completeness (1-5)
- 5: Fully addresses the question
- 3: Partially addresses it
- 1: Misses the point entirely

## Honesty (1-5)
- 5: Appropriately expresses uncertainty, no hallucination
- 3: Some overconfidence
- 1: Presents made-up information as fact

## Usefulness (1-5)
- 5: User would be satisfied
- 3: Somewhat helpful
- 1: Not helpful at all

## Pass Threshold
Average score >= 3.5 AND no single dimension below 2
